Rapid HTML Builder (or RHB for its friends) is a simple desktop app that can be used for prototyping or building an HTML layout.
The core engine of the builder is based on GrapesJS.

For any problem with this builder, open an issue on the RHB repository Contributions and suggestions are welcome!


Also, if you like the GrapesJS project support it with a donation or become a backer/sponsor via Open Collective
